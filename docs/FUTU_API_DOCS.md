# Futu OpenAPI Documentation

> **This file has been removed from the repository.**
> The full official documentation (~13,300 lines) was previously included here as a derivative work.
> It has been removed to comply with Futu's copyright and to keep this repository lean.

## Official Documentation

The authoritative Futu OpenAPI documentation is maintained by **Futu Holdings Limited** at:

- **Main docs:** https://openapi.futunn.com/
- **Hong Kong:** https://www.futunn.com/hk/openapi
- **API Reference:** https://openapi.futunn.com/futu-api-docs/

## Why This File Was Removed

The previous copy of Futu's documentation (~13,300 lines) was a derivative work of
Futu Holdings Limited's copyrighted material. Redistributing it without explicit
permission created legal risk. The official docs are freely available online and are
regularly updated.
